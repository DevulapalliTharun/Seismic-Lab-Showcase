# 🌍 Seismic-Lab-Showcase | Intelligence & Risk Modeling Portal

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_svg.svg)](https://seismic-os-djnkn4ggm9h6qcdga8qsfk.streamlit.app/)

**Seismic-Lab-Showcase** is a high-fidelity web-native platform developed for **NITK Surathkal** as part of the Data Driven Modeling (DDM) course. It showcases an end-to-end data science lifecycle—from raw seismic ingestion to distributed big data hazard assessment.

## 🛠️ Mission Modules

### 1. 🏗️ Data Engine (Lab 01)
*Foundational Data Engineering & Hybrid Storage*
- **ETL:** Parsing JSON, XML, HTML, and Binary seismic streams.
- **Cleaning:** Automated mean-imputation and regex-based feature extraction.
- **Storage:** Implementation of both Relational (**SQLite3**) and NoSQL (**MongoDB Atlas**) clusters.
- **Optimization:** Vectorized NumPy SIMD operations for O(1) analytics.

### 2. 🧠 Seismic AI (Lab 02)
*Predictive Modeling & Tectonic Intelligence*
- **Supervised Learning:** OLS Linear Regression for magnitude prediction.
- **Unsupervised Learning:** K-Means++ clustering (k=3) identifying global tectonic boundaries.
- **Preprocessing:** High-dimensional One-Hot Encoding (54k+ dimensions).

### 3. 🌩️ MapReduce Pipeline (Big Data)
*Distributed Hazard Assessment*
- **Framework:** Python **mrjob** for parallel processing.
- **Risk Index Model:** Composite scoring based on Frequency (50%), Magnitude (30%), and Intensity (20%).
- **Visualization:** 3D Geospatial Hexagon mapping using **Pydeck WebGL**.
